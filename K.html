Jawaban No 1

1.Web Server Web server merupakan software atau program komputer yang bertugas menerima request dari client (browser) melalui protocol HTTP atau HTTPS. Web server juga yang akan mengirimkan kembali hasil berupa halaman website ke browser client. Contoh web server adalah Apache, Nginx, IIS.

2. Domain Domain adalah alamat unik yang menjadi identitas dan digunakan user untuk mengakses sebuah website. Misalnya www.contoh.com. Domain ini akan diarahkan ke alamat IP web server tempat website di-hosting.

3. Hosting Hosting adalah tempat penyimpanan file-file website (HTML, CSS, Gambar, PHP, MySQL, dll) yang akan ditampilkan ke browser. Hosting inilah yang memuat web server tempat website diakses oleh client.

Jadi ketiga komponen tersebut saling berhubungan:

1. Browser client mengakses website lewat domain.

2. Domain mengarah ke hosting tempat website disimpan.

3. Web server di hosting memproses request dari client lalu mengirimkan kembali halaman website.
